# Air Inequity Index, Netherlands

![Python](https://img.shields.io/badge/python-3.10%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Data](https://img.shields.io/badge/data-Sentinel--5P%20%7C%20CAMS%20%7C%20Eurostat-orange)

A second iteration of the [EU Big Data Hackathon 2025 NSI_NL project](https://github.com/eurostat/eubd2025_results/tree/main/NSI_NL) funded by Eurostat. 

Combines satellite air quality data with Eurostat regional GDP to compute an **Air Inequity Index (AII)** — a measure of how economically vulnerable regions experience disproportionate air pollution exposure across Netherlands NUTS-3 regions.

---

## Quick start

```bash
pip install -r requirements.txt
cd dashboard
python app.py          # → http://localhost:8050
```

> The dashboard requires `data/processed/final.csv` and `data/processed/nl_nuts3.geojson`.
> Run the data pipeline below to generate them, or request the processed files separately.

## Data pipeline

### Step 0 — Set API keys

Copy the example config and fill in your credentials:
```bash
cp scripts/config.py.example scripts/config.py
```

**Copernicus Data Space Ecosystem (ESA)** — edit `scripts/config.py`:
```python
SH_CLIENT_ID     = "your-client-id"
SH_CLIENT_SECRET = "your-client-secret"
```
Register free at https://dataspace.copernicus.eu → User settings → OAuth clients → Create new client.

**Copernicus ADS (PM2.5)** — create `~/.cdsapirc`:
```
url: https://ads.atmosphere.copernicus.eu/api
key: <your-key>
```
Get your key at https://ads.atmosphere.copernicus.eu → My account → API key.

### Step 1 — Run scripts in order

```bash
python scripts/03_fetch_eurostat.py      # no key — run this first
python scripts/01_fetch_sentinel.py      # Sentinel Hub key required
python scripts/02_fetch_pm25_cds.py      # Copernicus ADS ~/.cdsapirc required
python scripts/04_merge_and_clean.py
python scripts/05_compute_index.py
python scripts/06_verify_output.py       # checks final.csv is valid
```

| Script | What it does | Runtime |
|---|---|---|
| `03_fetch_eurostat.py` | NUTS-3 GeoJSON + GDP + population | ~1 min |
| `01_fetch_sentinel.py` | NO2, SO2, CO, O3, HCHO from Sentinel-5P | Several hours |
| `02_fetch_pm25_cds.py` | PM2.5 from CAMS EAC4 global reanalysis | ~30 min |
| `04_merge_and_clean.py` | Merge all sources, fill gaps, add season | <1 min |
| `05_compute_index.py` | Quality scores, weights, compute AII | <1 min |
| `06_verify_output.py` | Sanity checks on final.csv | <1 min |

> **Tip:** Test the Sentinel Hub script with one pollutant and year before the full run:
> ```bash
> python scripts/01_fetch_sentinel.py --pollutants NO2 --years 2023
> ```
> The script saves progress after every row and can be safely interrupted and resumed.

## Methodology

### Quality scoring

Each pollutant concentration is converted to a **quality score (1–6)** using WHO/EPA thresholds:

| Pollutant | Thresholds | Unit |
|---|---|---|
| PM2.5 | 10, 15, 20, 30, 50 | µg/m³ |
| NO2   | 50, 100, 200, 500, 1000 | µg/m² |
| SO2   | 50, 100, 150, 200, 300 | µg/m² |
| O3    | 0.05, 0.10, 0.15, 0.20, 0.30 | mol/m² |
| CO    | 0.01, 0.02, 0.03, 0.05, 0.10 | mol/m² |
| HCHO  | 5, 10, 15, 20, 30 | µg/m² |

### Seasonal weights

Scores are multiplied by health-based seasonal weights and summed into a **Pollution Index**.
The weights reflect which pollutants dominate health risk per season (e.g. PM2.5 peaks in winter from heating; O3 peaks in summer from photochemistry):

| Pollutant | Winter | Spring | Summer | Autumn |
|---|---|---|---|---|
| PM2.5 | 0.40 | 0.36 | 0.25 | 0.35 |
| NO2   | 0.25 | 0.22 | 0.15 | 0.23 |
| O3    | 0.10 | 0.15 | 0.30 | 0.15 |
| SO2   | 0.12 | 0.12 | 0.05 | 0.12 |
| CO    | 0.06 | 0.07 | 0.10 | 0.07 |
| HCHO  | 0.07 | 0.08 | 0.15 | 0.08 |

### Air Inequity Index

```
GDP_Normalized     = 1 − (GDP_per_capita − min) / (max − min)   [inverted: low GDP → 1]
Air_Inequity_Index = Pollution_Index × GDP_Normalized
```

High AII = high pollution + low economic resources = most inequitable exposure.

## Dashboard features

- **Choropleth map** — NL NUTS-3 coloured by AII, Pollution Index, or GDP per capita

- **Time series** — monthly AII with range slider + 1Y/3Y/All buttons; toggle stacked pollutant contributions; syncs with map hover
